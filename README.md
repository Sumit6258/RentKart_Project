Rentkart is a full-stack rental ecommerce platform built using Django, Django REST Framework, AngularJS and Bootstrap.

## Features
- OTP-based authentication using Telegram
- Category & product browsing
- Subscription-based renting
- Admin & customer roles
- Image uploads
- AngularJS frontend

## Tech Stack
- Backend: Django, DRF
- Frontend: AngularJS, Bootstrap
- Database: SQLite (Dev)
- Auth: OTP via Telegram

## Setup
1. Create virtual environment
2. Install requirements
3. Run migrations
4. Start backend & frontend servers

cd frontend
python -m http.server 5500
